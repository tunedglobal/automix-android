AutomixSDKPluginMetadataDictionaryEntryModel
Link copied to clipboard
data class AutomixSDKPluginMetadataDictionaryEntryModel(val type: String?, val meta: String?)

AutomixSDKPluginMetadataDictionaryEntryModel models metadata of a track traditionally obtained by running plugins on it. Key and BPM are recommented for best mixing. The Type entry for bpm and key are "key" and "bpm" (case sensitive) respectively, the values need to be formatted as strings, and the key needs to be in OpenKey format.
