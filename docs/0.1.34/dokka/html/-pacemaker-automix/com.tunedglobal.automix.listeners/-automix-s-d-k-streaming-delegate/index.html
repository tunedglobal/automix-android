AutomixSDKStreamingDelegate

interface AutomixSDKStreamingDelegate

AutomixSDKStreamingDelegate protocol defines the methods to be implemented by a class tasked with providing the Automix SDK the information it requires to play content. See TunedDemoStreamingHandler for an example of an object implementing this protocol.
